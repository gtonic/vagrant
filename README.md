# vagrant
Tom's development Box

## Prerequisites

- vagrant
- virtualbox (default vm) guest plugin: `vagrant plugin install vagrant-vbguest`

## Installed software

- curl
- docker
- dotnet
- eclipse
- gradle
- java 8
- java 14
- nodejs
- oc client
- vscode
- wget
